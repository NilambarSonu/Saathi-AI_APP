# ⚡ Gemini CLI + 1,325 Skills - Quick Start

## 🚀 Get Started in 2 Minutes

```powershell
# 1. Close all terminals (hard close)
# 2. Open NEW PowerShell
# 3. Navigate to project
cd "C:\Your\Project"

# 4. Start Gemini with skills
start-gemini

# 5. In Gemini, use a skill
Use @brainstorming to plan this feature
Use @debugging-strategies to fix this error
Use @security-auditor to review this code
```
